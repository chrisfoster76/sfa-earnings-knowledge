# Individualised Learner Record (ILR)

**Source:** https://guidance.submit-learner-data.service.gov.uk/

## What It Is

The ILR is the data return that training providers submit to DfE to evidence funded learning activity. It is the primary mechanism by which providers claim funding for apprenticeships, T Levels, adult education, and other programmes.

Submissions are made via the **Submit Learner Data (SLD)** service.

---

## Documentation Structure

The guidance site is organised by **academic year** (1 Aug – 31 Jul). Each year has:

| Document | Purpose |
|----------|---------|
| ILR Specification | Field-level data definitions and business rules |
| Validation Rules | Automated checks applied when a file is submitted |
| Collection Timetable | Deadlines for each data return within the year |
| Migration Specification | Changes from the prior year's spec |
| Schema & Sample Files | XSD schema and example ILR XML files |
| Privacy Notice | GDPR/data handling obligations |
